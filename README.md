# Go-opentelemetry App

## Steps to Run

1. **Clone the repository**:  
   ```bash
   git clone https://github.com/yogesh895/go-opentelemetry.git
   cd go-opentelemetry
   ```

2. **Install dependencies**:  
   ```bash
   go mod tidy
   ```

3. **Run the application**:  
   ```bash
   go run main.go
   ```
